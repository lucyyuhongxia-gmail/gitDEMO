Troubleshooting

Find the message you see, then apply the fix.

“nothing to commit, working tree clean”

This is not an error. It means Git sees no changes to commit.

“fatal: not a git repository”

  • Make sure you are in the project folder (check your terminal path).
  • If needed, initialize a repo: git init.

“rejected: non-fast-forward” when pushing

  • Someone else pushed first. Pull, resolve conflicts if needed, then push again.
  • Typical fix: git pull then git push.

“detached HEAD”

  • You checked out a commit instead of a branch.
  • Fix: git switch main (or your branch name).
