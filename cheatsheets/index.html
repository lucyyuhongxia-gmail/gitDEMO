Cheat Sheets

Fast reference for classwork.

Daily workflow (CLI)

git status
git add .
git commit -m "Message"
git push

Branch workflow (CLI)

git switch -c feature/name
git add .
git commit -m "Add feature"
git push -u origin feature/name

Undo safely (CLI)

# unstage (keep edits)
git restore --staged file

# discard edits (careful)
git restore file

# undo a pushed commit safely
git revert <sha>

Printable tip: use your browser’s print dialog.
