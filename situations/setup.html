Setup checklist

Do this once per laptop/user account.

Objective

Install Git, set your identity, and connect your computer to GitHub.

Command Line

  1. Check Git is installed:
git --version
  1. Set your identity (use your real name):
git config --global user.name "First Last"
git config --global user.email "you@example.com"
  1. Confirm your settings:
git config --global --list
Recommendation: Use HTTPS first (simpler). Use SSH later if you already know it.

VS Code

  1. Open VS Code → View → Terminal.
  2. Run the same Git commands above in the terminal.
  3. Open Source Control panel to confirm VS Code detects Git.

GitHub

  1. Create a GitHub account (if you don’t have one).
  2. Verify your email (GitHub requires this for pushes).
  3. Create a new repository for your CS project (public or private as your teacher requires).

Check you succeeded

  • git --version prints a version number.
  • git config --global --list shows your name and email.
  • VS Code Source Control shows Git controls when a repo is open.

Troubleshooting

  • “git: command not found” → install Git (Windows: Git for Windows; macOS: Xcode Command Line Tools).
  • Push fails with auth error → use a GitHub Personal Access Token (PAT) for HTTPS.
  • Commits show the wrong name/email → update the global config and try again.
