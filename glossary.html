Glossary

Quick definitions (school-friendly).

Repository (repo)

A project folder tracked by Git, including its history.

Commit

A snapshot of your project with a message describing what changed.

Staging area

The “next commit” box. You choose which changes go into the next commit.

Branch

A named line of development. Use branches to work on features without breaking main.

Remote

A copy of your repo hosted elsewhere (often GitHub). origin is the common default name.

HEAD

A pointer to where you are currently “standing” in Git history (usually the latest commit of your branch).
