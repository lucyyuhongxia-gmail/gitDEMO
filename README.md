# Git for IB DP CS (Static Site)

This repo hosts a simple static website for IB DP Computer Science students to learn Git in common situations using:

- Command line Git
- VS Code Source Control + terminal
- GitHub

## Local preview

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

1. GitHub repo → **Settings** → **Pages**
2. **Build and deployment** → Source: **Deploy from a branch**
3. Branch: `main` and Folder: `/ (root)`
4. Save, then open the URL GitHub shows.
